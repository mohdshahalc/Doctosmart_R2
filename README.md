# Doctosmart_R2

A modern, production-ready React frontend application for patient management, built with Vite and Tailwind CSS.

## Features
- Server-side data fetching mapped accurately to the API.
- Dashboard with dynamic stat cards and sleek gradients.
- Responsive, accessible table views for patient records.
- Professional, minimalist UI with Inter font integration.
- Instant CSV exporting.
- Client-side search and gender filtering.

## Tech Stack
- **React 19**
- **Vite**
- **Tailwind CSS v3**
- **Lucide React** (Icons)
- **Axios** (API requests)

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
